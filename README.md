# README #

Proyecto de prueba para consumir [servicios de Pokemon](https://pokeapi.co/)
Este desarrollo tiene lo siguiente:
  * CleanArquitecture con MVP
  * Base de datos Room
  * RxAndroid
  * SharedPreference
  * Glide
  * Manejo de servicios rest a modo Offline

Los datos de login son:

* Usuario: *pokemon*
* Password: *pokemon*
